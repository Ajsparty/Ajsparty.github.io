# Aidan Swenson Portfolio

## Personal Portfolio Website

This repository hosts my professional portfolio website, built as a single-page application using HTML, Tailwind CSS, and vanilla JavaScript. It showcases my work as a Computational Data Science student at Michigan State University, highlighting intermediate-level projects in machine learning, data visualization, and full-stack development.

## Live Site

Once GitHub Pages is enabled, the site will be available at: https://ajsparty.github.io

### How to Enable GitHub Pages
1. Go to the repository settings.
2. Navigate to **Pages** in the left sidebar.
3. Under "Build and deployment", select **Source** > **Deploy from a branch**.
4. Choose branch `main` and folder `/ (root)`.
5. Click **Save**.

The site will be live within a few minutes.

## Customization
- Edit `index.html` to update personal information, projects, or styling.
- Add your resume PDF in the `/assets` folder (create if needed) and update the download link.
- For advanced features, consider migrating to a framework like Next.js or Astro.
